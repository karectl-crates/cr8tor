# Cr8tor

CR8tor is a end-to-end governance framework for the secure creation, management and automated provision of federated data projects within kubernetes-first trusted research environments. For an overview of CR8tor capabilities, see our walkthrough video [here](https://www.youtube.com/watch?v=aRrZpoF2YeA).  


For further technical documentation and developer gudiance go to the CR8tor informatio portal [here](https://karectl-crates.github.io/cr8tor/).
